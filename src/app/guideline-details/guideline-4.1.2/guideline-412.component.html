4.1.2 - Name, Role, Value Official 4.1.2 WCAG Docs

This success criterion aims to help people using assisstive technology understand all components

Who does this benefit?

  • People using assistive technology, such as screen readers, screen magnifiers, and speech recognition software

How to meet this criterion

  • Using aria-label to provide an invisible label where a visible label cannot be used
  • Using aria-labelledby to provide a name for user interface controls
